# 更新日志

所有重要的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-02-04

### 新增
- 初始版本发布
- 支持英文→中文 LaTeX 论文翻译
- 自动保留所有 LaTeX 格式和命令
- 支持 OpenAI compatible API
- 实时显示翻译进度
- 断点续传功能
- 自动生成中文主文件
- 自动编译中文 PDF
- 中文字体自动配置

### 特性
- 使用段落级别翻译，保证上下文连贯
- 专业术语自动标注英文
- 跳过已翻译文件，节省成本
- 详细的进度显示和日志
- 灵活的配置选项

### 文档
- 完整的 README
- 快速开始指南（QUICKSTART）
- API 配置示例（.env.example）
- 详细的 prompt 说明

### 已知限制
- 当前仅支持英文→中文方向
- 需要 xelatex 支持中文编译
- 依赖中文字体
